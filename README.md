# Azure Python Function sample

This sample shows how to perform text processing and entity recognition using nltk and spaCy in an Azure Python Function.

## How to use

[This tutorial](https://medium.com/@k.giguz/getting-started-with-azure-python-functions-abd4d88706f7?source=friends_link&sk=ff268566b61cea31a5dc3a43f6316e3f) will walk you through how to get started with Azure Python Functions and how to publish a Functions project to Azure.
